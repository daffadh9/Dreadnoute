
# DreadNoute Product Roadmap (12 Months)

Author: Daffa Dhiyaulhaq Khadafi

## Year 1 Objective
Tujuan tahun pertama bukan membuat superapp penuh, tetapi membuktikan bahwa DreadNoute layak hidup di pasar.

Target utama:
- Mendapatkan user pertama
- Membangun engagement
- Membuktikan retention
- Menguji monetisasi awal

---

## Phase 1 — Foundation (Month 1–2)

Fokus: Infrastruktur dasar platform

Fitur:
- Authentication system
- User profile
- Database schema
- Design system horror UI
- Basic Ghost Archive structure
- Basic Ghost Card structure

Outcome:
Fondasi platform stabil untuk pengembangan fitur berikutnya.

---

## Phase 2 — Knowledge Engine (Month 3–4)

Fokus: Akuisisi user

Fitur:
- Ghost Archive pages
- Category & tag system
- Search system
- Lore entries
- Visual archive layout

Outcome:
Mulai mendapatkan traffic dari search dan discovery.

---

## Phase 3 — Participation Engine (Month 4–6)

Fokus: User engagement

Fitur:
- Real or Fake investigation feature
- Media upload
- Voting system
- Comments
- Investigation result pages

Outcome:
User mulai berinteraksi dan membuat konten sendiri.

---

## Phase 4 — Retention System (Month 5–7)

Fokus: Membuat user kembali

Fitur:
- Ghost Cards (G-Collector)
- Card rarity
- Card unlocking
- Profile showcase
- Collection progress

Outcome:
User memiliki motivasi untuk kembali ke platform.

---

## Phase 5 — Economy Layer (Month 7–9)

Fokus: Sistem ekonomi platform

Fitur:
- DreadCoin wallet
- Reward system
- Activity rewards
- Card packs / collectible drops

Outcome:
Platform mulai memiliki ekonomi internal.

---

## Phase 6 — Early Monetization (Month 9–12)

Fokus: Membuktikan willingness to pay

Fitur:
- Premium membership (Ghost Pass)
- Limited digital collectibles
- Creator tipping
- Early creator onboarding

Outcome:
Revenue signal pertama.

---

## Success Metrics

Target tahun pertama:

Users: 10,000 – 50,000  
Active users: 2,000 – 10,000  

Key signals:
- retention
- engagement
- first revenue
