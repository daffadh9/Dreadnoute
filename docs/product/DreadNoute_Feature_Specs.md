
# DreadNoute Feature Specifications

## Ghost Archive

Purpose:
Knowledge hub for horror lore.

Features:
- entity pages
- tags
- categories
- related content
- media gallery

---

## Real or Fake

Purpose:
Community investigation system.

Features:
- media upload
- voting
- comments
- investigation result

---

## Ghost Cards

Purpose:
Gamified collection system.

Features:
- rarity levels
- card sets
- profile showcase

---

## DreadCoin

Purpose:
Internal platform currency.

Uses:
- rewards
- card packs
- premium features
