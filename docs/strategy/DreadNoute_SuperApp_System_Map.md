
# DreadNoute SuperApp System Map

Author: Daffa Dhiyaulhaq Khadafi

## Core Ecosystem Loop

Discovery → Knowledge → Participation → Collection → Identity → Economy → Creator → Community → Discovery

---

## Discovery Layer
Purpose: bring new users into the ecosystem.

Features:
- Ghost Archive
- Blog
- Horror Wiki
- Trailer discovery

---

## Knowledge Layer
Users explore horror knowledge.

Features:
- Ghost Archive
- Horror Wiki
- Lab Ghost

Goal:
Become a global horror knowledge authority.

---

## Participation Layer
Users start interacting.

Feature:
Real or Fake

Flow:
User upload → Community vote → Discussion → Investigation result

---

## Collection Layer
Users receive rewards.

Feature:
Ghost Cards (G‑Collector)

Examples:
- Investigation reward
- Event reward
- Rare card drops

Goal:
Retention.

---

## Identity Layer
User builds status.

Features:
- Profile
- Reputation
- Card collection
- Achievements

---

## Economy Layer
Internal economy begins.

Features:
- DreadCoin
- Obsidian
- Digital collectibles
- Premium access

---

## Creator Layer
Creators enter the ecosystem.

Tools:
Ghost Creator Toolkit

Creators produce:
- Stories
- Artwork
- Investigations
- Collectibles

---

## Community Layer
Community expansion.

Features:
- Events
- Challenges
- Competitions
- Seasonal activities

---
