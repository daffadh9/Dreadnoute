# DreadNoute Branding Assets

This folder contains references and metadata for the DreadNoute brand.

## Visual Identity
- **Primary Color**: #FF0000 (Neon Crimson)
- **Background**: #050505 (Obsidian Black)
- **Typography**: Cinzel (Heading), Inter (Body)
- **Aesthetic**: Minimalist Horror, Gritty, immersive.

## Assets
- Logos and icons are stored in `public/branding/` for web accessibility.
- `logo.jpg`: Main app logo (provided by user).
- `favicon.ico`: App favorite icon.

## App Icons
Icons used throughout the app are from `lucide-react`, styled with a stroke width of 1.5 for a clean, professional look.
